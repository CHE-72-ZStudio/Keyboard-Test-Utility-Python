# 「鍵盤檢測工具程式（Python）」的用戶手冊
## User Manual for Keyboard Test Utility (Python)

## 詳細使用流程與運行步驟
1. 將從 Realease 下載出來的壓縮包解壓縮。
2. 執行解壓縮後的 KTU_CHE72_ZStudio.exe。
3. 按下你想測試的鍵盤按鍵，此時視窗中應當出現您目前按下的按鍵內容與按下次數。
4. 如果測試完成，要關閉本程序，只需按下右上角的關閉按鈕即可。
